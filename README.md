# Spring Boot-Chat

本项目是基于Netty实现的一个实时通讯系统，前端使用了LayIM组件。



| 技术栈                           |
| -------------------------------- |
| Spring Boot、Spring MVC、MyBatis |
| MySQL、Redis                     |
| Netty、WebSocket                 |
| Docker、Nginx、FastDFS           |
| LayIM                            |

## 实现功能

### 客户端

- [ ] QQ授权登陆（审核没通过，只能用于本地测试）
- [x] 单聊
- [x] 群聊
- [x] 发送图片/文件
- [x] 查看聊天记录
- [x] 创建群聊
- [x] 搜索好友/群聊
- [x] 添加好友/群聊
- [x] 上下线提醒
- [x] 在线/隐身
- [x] 单聊/群聊的离线消息签收

### 管理端

- [x] 查看用户/群聊的聊天记录
- [x] 重置用户密码
- [x] 禁用用户（强制下线）
